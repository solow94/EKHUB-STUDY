W3C Invites Implementations of CSS Text Decoration Module Level 3 Tue Jul 03 2018 The CSS Working Group invites implementations of an updated Candidate Recommendation of CSS Text Decoration Module Level 3. This module contains the features of CSS relating to text decoration, such as underlines, text shadows, and emphasis marks. CSS is a language for describing the rendering of structured documents (such as HTML and XML) on screen, on paper, in speech, etc.
